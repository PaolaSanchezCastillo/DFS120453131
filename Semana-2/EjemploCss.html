Modulo HTML-CSS

¿Qué es HTML?

HTML son las siglas de HyperText Markup Language, que constituye un estándar asumido y reconocido a nivel mundial. ... Es decir, con el lenguaje HTML se crean las páginas web, pero no su diseño, mientras que con CSS (un lenguaje de hojas de estilos), se le da a esos contenidos una apariencia atractiva para el usuario.

¿Qué es CSS?

CSS, en español «Hojas de estilo en cascada», es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.
